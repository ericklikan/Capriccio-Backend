# HTN2019 Flask backend

This is a Flask backend for our app using the blueprints.

## Quickstart

```bash
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt

chmod +x run.sh
./run.sh
```

## starting up

```bash
➜ python3 -m venv venv
➜ . venv/bin/activate
➜ pip3 install -r requirements.txt

➜ export FLASK_APP=api
➜ export FLASK_ENV=development

➜ flask run
```
